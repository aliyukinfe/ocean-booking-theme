# Ocean Booking Theme

Custom WordPress theme for the Ocean Booking website.

## WP Pusher

Install this repository as a **theme** in WP Pusher. The theme files live at the repository root and do not require a build step.

Required companion plugin: `ocean-booking-core`.

## Setup

1. Install and activate the Ocean Booking Core plugin.
2. Install and activate this theme.
3. Assign menus in **Appearance > Menus**.
4. Edit home page content and hero image in **Appearance > Customize > Home Page Content**.
5. Use Polylang, WPML, or TranslatePress for translations.
